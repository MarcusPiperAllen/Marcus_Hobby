## ğŸ”„ GitHub â†” Replit Workflow

### Initial Setup (One-Time)

1. **Fork or Clone this Repository**
   ```bash
   git clone https://github.com/MarcusPiperAllen/Marcus_Hobby.git
   cd Marcus_Hobby
   ```

2. **Connect to Replit**
   - Go to [Replit](https://replit.com)
   - Click "Create Repl"
   - Select "Import from GitHub"
   - Enter: `https://github.com/MarcusPiperAllen/Marcus_Hobby`
   - Click "Import from GitHub"

### Syncing Updates from GitHub to Replit

**Method 1: Using Replit's Git Integration (Recommended)**

1. Open your Repl
2. Click the **Version Control** icon (Git branch icon) in the left sidebar
3. Click **Pull** to fetch the latest changes from GitHub
4. Your Replit site will automatically update

**Method 2: Using Replit Shell**

1. Open the **Shell** tab in Replit
2. Run these commands:
   ```bash
   git pull origin main
   ```
3. Refresh your preview to see changes

### Pushing Local Changes to GitHub

From your local development environment:

```bash
# Stage all changes
git add .

# Commit with a descriptive message
git commit -m "Your commit message here"

# Push to GitHub
git push origin main
```

After pushing to GitHub, follow the "Syncing Updates" steps above to pull changes into Replit.
